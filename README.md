
# Student Lost and Found Management System

## Project Description
A web-based system that helps students report, search, and recover lost items within the university campus.

## Technology Stack
- Backend: ASP.NET Core MVC (C#)
- Database: MySQL
- Frontend: HTML, CSS, Bootstrap, JavaScript
